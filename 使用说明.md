# 飞书RPA监控系统 - 使用说明

## 🚀 快速开始

### 1. 启动程序

双击 **`飞书RPA监控系统.exe`** 启动

程序会自动：
- 创建必要的文件夹（`data`、`reports`）
- 初始化数据库
- 启动Web服务器

看到以下提示表示启动成功：
```
====================================================
  飞书RPA监控系统 Web服务
====================================================
  访问地址: http://127.0.0.1:8888
  API文档: http://127.0.0.1:8888/docs
====================================================
```

### 2. 打开浏览器

启动后，打开浏览器访问：**http://127.0.0.1:8888**

---

## ⚙️ 首次配置

### 方法1：通过配置页面（推荐）

1. 访问：http://127.0.0.1:8888/#/config
2. 填写以下配置：

#### AI配置（必填）
- **API Key**: DeepSeek的API密钥
- **API Base**: `https://api.deepseek.com`
- **模型**: `deepseek-chat`

#### 飞书配置（选填）
- **App ID**: 飞书应用的App ID
- **App Secret**: 飞书应用的App Secret
- **Chat ID**: 飞书群聊的Chat ID

#### 邮箱配置（选填）
- **SMTP服务器**: 如 `smtp.163.com`
- **端口**: 如 `465`
- **邮箱账号**: 发送邮件的邮箱
- **授权码**: 邮箱的SMTP授权码
- **业务负责人邮箱**: 多个用逗号分隔
- **技术运维邮箱**: 多个用逗号分隔

3. 点击"保存配置"
4. **重启程序**让配置生效

---

### 方法2：手动创建配置文件

在EXE同目录创建 `.env` 文件，内容如下：

```env
# ========== AI配置 ==========
AI_API_KEY=你的DeepSeek API Key
AI_API_BASE=https://api.deepseek.com
AI_MODEL=deepseek-chat

# ========== 飞书配置 ==========
FEISHU_APP_ID=你的App ID
FEISHU_APP_SECRET=你的App Secret
FEISHU_CHAT_ID=群聊的Chat ID

# ========== 邮箱配置 ==========
SMTP_HOST=smtp.163.com
SMTP_PORT=465
SMTP_USER=你的邮箱@163.com
SMTP_PASSWORD=你的SMTP授权码
BUSINESS_EMAILS=业务负责人1@example.com,业务负责人2@example.com
TECH_EMAILS=技术运维1@example.com,技术运维2@example.com

# ========== 企业微信配置（可选）==========
WECOM_WEBHOOK=企业微信机器人Webhook地址

# ========== 定时任务配置 ==========
DAILY_RUN_HOUR=20
DAILY_RUN_MINUTE=0
```

保存后重启程序。

## 📱 功能使用

### 智能对话

访问：http://127.0.0.1:8888/#/chat

**示例对话：**

```
你：分析今天的日志
AI：好的，开始分析...

你：每天下午4点帮我分析日志并发邮件
AI：收到！已创建定时任务...

你：生成今天的报告并发给领导
AI：正在生成报告...
```

### 数据分析

访问：http://127.0.0.1:8888/#/analysis

选择日期范围，点击"开始分析"，查看分析结果。

### 定时任务

访问：http://127.0.0.1:8888/#/tasks

- 查看所有定时任务
- 启用/停用任务
- 查看执行历史

### 历史报告

访问：http://127.0.0.1:8888/#/reports

- 查看所有生成的报告
- 下载报告文件

## 📂 文件结构

程序运行后会在EXE同目录创建：

```
飞书RPA监控系统.exe      # 主程序
.env                       # 配置文件（手动创建或配置页面生成）
data/                      # 数据目录
  └── rpa_monitor.db       # SQLite数据库（自动创建）
reports/                   # 报告输出目录（自动创建）
  └── *.txt                # 生成的报告文件
```

## ❓ 常见问题

### Q: 启动后浏览器无法访问？
A: 检查是否有其他程序占用8888端口，或者防火墙拦截。

### Q: 配置保存后不生效？
A: 需要重启程序才能加载新配置。

### Q: 飞书推送失败？
A: 检查App ID、App Secret、Chat ID是否正确，以及应用是否有发送消息权限。

### Q: 如何获取DeepSeek API Key？
A: 访问 https://platform.deepseek.com/ 注册账号后在API Keys页面创建。

### Q: 如何获取飞书Chat ID？
A: 在飞书群里发送任意消息，通过飞书API获取chat_id。

### Q: 定时任务不执行？
A: 确保程序持续运行，不要关闭EXE的控制台窗口。

### Q: 如何关闭程序？
A: 在控制台窗口按 `Ctrl+C` 或直接关闭窗口。
