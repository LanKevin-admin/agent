## Skill: 文件校验

### 能力
- 校验文件是否存在
- 检查文件修改时间（超过1天告警）
- 校验文件名日期与查询范围是否匹配

### 可用工具
- `validate_file` - 单个文件校验
- `validate_multiple_files` - 批量校验

### 重要参数
- `expected_date_range` - **必须传入**，格式：`YYYY-MM-DD,YYYY-MM-DD`
- 从RPA日志的"查询范围"字段提取

### 文件名日期格式支持
- 连续8位数字：`20260501-20260630`
- 带横线：`2026-05-01` 到 `2026-06-30`

### 校验规则
- 文件不存在 → 失败
- 文件修改时间超过1天 → 警告（可能是旧文件冒充）
- 文件名日期与查询范围不匹配 → 失败
