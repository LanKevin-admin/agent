## 📝 添加新功能

### 后端：添加新的Skill

**1. 创建Skill类**

在 `skills/` 目录创建新文件：

```python
# skills/my_new_skill.py
from skills.base_skill import BaseSkill

class MyNewSkill(BaseSkill):
    """新技能描述"""
    
    def __init__(self):
        super().__init__(name="my_new_skill")
    
    @BaseSkill.tool("my_function", "函数描述")
    def my_function(self, param1: str, param2: int) -> dict:
        """
        工具函数
        
        Args:
            param1: 参数1描述
            param2: 参数2描述
            
        Returns:
            {"success": True, "data": ...}
        """
        # 实现业务逻辑
        return {"success": True, "data": "result"}
```

**2. 注册Skill**

在 `agent/skill_based_agent.py` 中注册：

```python
from skills.my_new_skill import MyNewSkill

def _register_skills(self):
    # ... 其他Skill
    self.skill_manager.register_skill(MyNewSkill())
```

**3. 测试**

通过智能对话测试新功能。


**2. 添加路由**

在 `web/src/main.js` 中添加路由：


## 🔧 修改配置

### 添加新的环境变量

**1. 修改 `config/settings.py`**

```python
class MyNewConfig:
    """新配置类"""
    MY_SETTING = os.getenv("MY_SETTING", "default_value")

class AppConfig:
    # ... 其他配置
    mynew = MyNewConfig()
```

**2. 更新 `.env.example`**

```env
# ========== 新配置 ==========
MY_SETTING=your_value
```

**3. 使用配置**

```python
from config.settings import config

value = config.mynew.MY_SETTING
```

## 📋 代码规范

### Python

- 使用PEP 8规范
- 类名：大驼峰（`MyClass`）
- 函数名：小写下划线（`my_function`）
- 常量：大写下划线（`MY_CONSTANT`）
- 文档字符串：使用docstring

### JavaScript/Vue

- 使用ES6+语法
- 组件名：大驼峰（`MyComponent.vue`）
- 变量名：小驼峰（`myVariable`）
- 使用Composition API + `<script setup>`
