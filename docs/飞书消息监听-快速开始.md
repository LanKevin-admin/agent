# 🎉 飞书消息监听功能 - 快速总结

## ✨ 新功能

**自动监听并回复飞书群@消息**

在飞书群里@机器人，10秒内自动回复！

---

## 🎮 用户体验

### 使用前
```
用户在群里：@机器人 帮我分析日志
（等待...没反应）
需要打开网页手动输入问题
```

### 使用后
```
用户在群里：@机器人 帮我分析日志
（10秒内）
机器人：好的，我来分析今天的日志数据...
[完整分析结果自动回复]
```

---

## 📋 核心特性

| 特性 | 说明 |
|------|------|
| **自动检测** | 每10秒检查一次群消息 |
| **智能识别** | 只响应@机器人的消息 |
| **防重复** | 同一条消息只回复一次 |
| **全功能** | 支持所有AI能力（分析、报告、定时任务等） |
| **后台运行** | 不阻塞主程序 |
| **异常处理** | 出错时回复提示，不影响继续监听 |

## 🎯 工作流程

```
启动程序
    ↓
后台线程每10秒执行
    ↓
检查最近1分钟的群消息
    ↓
找到@机器人的新消息
    ↓
提取用户问题
    ↓
agent.run(问题) → AI处理
    ↓
自动回复到群聊
    ↓
记录消息ID（防重复）
```

## 🔧 配置说明

### 默认配置

```python
interval = 10  # 检查间隔：10秒
max_length = 4000  # 回复长度限制
max_history = 1000  # 记录已处理消息数量
```

### 必需配置

在`.env`文件中：
```env
FEISHU_APP_ID=cli_xxx
FEISHU_APP_SECRET=xxx
FEISHU_CHAT_ID=oc_xxx
```
