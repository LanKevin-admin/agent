### 1. 邮件附件支持

**问题：** 报告生成TXT文件，但邮件发送时没有带附件

**解决方案：**

**A. 邮件发送器支持附件**
- 文件：`notify/email_sender.py`
- 增加`attachment_path`参数
- 使用`MIMEApplication`添加附件

**B. 工具函数支持附件**
- 文件：`agent/tools.py` - `tool_send_email`
- 增加`attachment_path`参数

**C. Skill定义支持附件**
- 文件：`skills/notification_skill.py`
- send_email工具增加`attachment_path`参数定义

**D. 系统提示词明确要求**
- 文件：`prompts/system_prompt.md`
- 第5条："发送邮件时必须带上附件"
- 示例：`send_email(recipient_type="business", subject="RPA日报", body=summary, attachment_path=detail_file)`

**使用示例：**
```python
# 生成报告
report = generate_executive_report(analysis_data)
# report返回: {"summary": "...", "detail_file": "./reports/xxx.txt", ...}

# 发送邮件（带附件）
send_email(
    recipient_type="business",
    subject="RPA每日运行报告",
    body=report["summary"],
    attachment_path=report["detail_file"]  # 关键：传递附件路径
)
```

### 2. 定时任务Cron表达式AI自动生成

**问题：** 用户创建定时任务时，AI不会自动计算Cron表达式

**解决方案：**

**文件：** `prompts/system_prompt.md` - 第7条

**新增内容：**

```markdown
7. **定时任务管理**（重要！）
   - **你需要自己计算Cron表达式**，不要让用户提供
   
   **Cron表达式生成规则**（5个字段：分 时 日 月 周）：
   - "每天18点" → `0 18 * * *`
   - "每天下午6点半" → `30 18 * * *`
   - "每天早上9点" → `0 9 * * *`
   - "每天9点和18点" → `0 9,18 * * *`
   - "每周一18点" → `0 18 * * 1`
   - "每周一三五18点" → `0 18 * * 1,3,5`（注意：1=周一，5=周五，0或7=周日）
   - "每周一到周五18点" → `0 18 * * 1-5`
   - "每月1号9点" → `0 9 1 * *`
   - "每月15号18点" → `0 18 15 * *`
   - "每30分钟" → `*/30 * * * *`
   - "每小时" → `0 * * * *`
```

**对话示例：**

**用户：** "每天下午4点帮我分析日志"

**AI思考过程：**
1. 下午4点 = 16:00
2. 每天 = `* * *`（日、月、周都是任意）
3. 生成Cron：`0 16 * * *`
4. 调用`create_scheduled_task(task_name="每日日志分析", cron_expression="0 16 * * *", ...)`

**AI回复：** "好的，定时任务已设置，以后每天下午4点我会自动帮你分析日志"

---

**用户：** "每周五下午6点发周报"

**AI思考过程：**
1. 周五 = 5（周一到周日是1-7，0和7都是周日）
2. 下午6点 = 18:00
3. 生成Cron：`0 18 * * 5`
4. 调用`create_scheduled_task(task_name="每周周报", cron_expression="0 18 * * 5", ...)`

**AI回复：** "搞定，每周五下午6点自动发周报"

### 3. 飞书推送说明

**当前状态：** 
- 飞书API只支持文本消息（`send_text`、`send_report`）
- 不支持文件上传（需要额外开发）

**推荐方案：**
- 飞书：发送文本摘要
- 邮件：发送摘要 + TXT附件（已支持）

**工作流：**
```
1. 生成报告 → {"summary": "...", "detail_file": "..."}
2. 发飞书 → send_feishu_message(summary)
3. 发邮件 → send_email(subject="...", body=summary, attachment_path=detail_file)
```
