## 🔧 常见问题

### Q1：双击没反应？

检查是否被杀毒软件拦截，需要添加信任。

---

### Q2：端口被占用？

如果8888端口被占用，可以创建一个批处理文件 `start.bat`：
```batch
@echo off
飞书RPA监控系统.exe --port 9999
pause
```

（注：需要修改代码支持命令行参数）

---

### Q3：如何部署到服务器？

1. 将整个 `dist/` 文件夹复制到服务器
2. 配置 `.env` 文件
3. 运行 `飞书RPA监控系统.exe`
4. 访问 `http://服务器IP:8888`

---

### Q4：如何后台运行？

安装 NSSM 或创建Windows服务：
```batch
# 使用NSSM（推荐）
nssm install "飞书RPA监控" "E:\path\to\飞书RPA监控系统.exe"
nssm start "飞书RPA监控"
```
