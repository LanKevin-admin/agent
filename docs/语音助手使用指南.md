# 语音助手功能说明

## 🎙️ 功能概述

**像打电话一样和AI助手对话！**

不用打字，直接说话：
- 🎤 点击麦克风，开始说话
- 🤖 AI实时理解你的需求
- ❓ 不理解会主动提问
- ✅ 确认后自动执行任务
- 🔊 结果语音播报

---

## 🌟 核心特性

### 1. 实时语音识别
- 中文语音识别
- 实时显示识别结果
- 支持长句识别

### 2. 智能理解
- AI理解你的意图
- 不确定时主动询问
- 需要确认时明确提问

### 3. 语音播报
- AI回复自动播报
- 可关闭语音播报
- 支持调节语速音量

### 4. 连续对话
- 开启连续模式
- 无需重复点击
- 自然对话体验

## 📱 使用场景

### 场景1：查看监控数据

```
你说：帮我看看今天的监控数据

AI：我来查询一下今天的监控数据...
（执行查询）
AI：今天共有1250条记录，其中成功1180条，失败70条。

你问：有什么异常吗？

AI：发现3个异常：
1. 上午10点某任务超时
2. 下午2点数据库连接失败
3. 下午4点API调用异常
```

### 场景2：创建定时任务

```
你说：每天下午5点给我发邮件报告

AI：我理解你想创建一个定时任务，每天17:00发送邮件报告，对吗？

你说：对的

AI：（执行创建）
好的，定时任务已创建成功！每天17:00会自动发送邮件报告。
```

### 场景3：修改配置

```
你说：把邮箱改成新的

AI：你想修改哪个邮箱配置？是SMTP发件箱还是收件人邮箱？

你说：SMTP发件箱

AI：请告诉我新的SMTP邮箱地址

你说：new@example.com

AI：（执行修改）
SMTP邮箱已更新为 new@example.com
```

## 🎮 操作指南

### 基础操作

1. **开始对话**
   - 点击麦克风图标
   - 看到"正在听..."提示
   - 开始说话

2. **停止录音**
   - 再次点击麦克风
   - 或者停止说话2秒自动结束

3. **查看历史**
   - 页面下方显示对话历史
   - 包含时间戳
   - 可滚动查看

### 高级功能

**连续对话模式：**
- 打开"连续对话模式"开关
- 无需重复点击麦克风
- AI回复完自动开始下一轮

**关闭语音播报：**
- 关闭"AI语音播报"开关
- 只显示文字，不播报
- 适合办公室等安静场所

## 💡 对话技巧

### 清晰表达

**❌ 不好的说法：**
- "看一下那个"
- "帮我弄一下"
- "改一下配置"

**✅ 好的说法：**
- "查看今天的监控数据"
- "创建每天17点发送邮件的定时任务"
- "修改SMTP邮箱为new@example.com"

### 分步确认

```
场景：创建复杂任务

你：创建一个定时任务

AI：好的，请告诉我具体要做什么？

你：每天下午5点分析日志并发邮件

AI：收到！要发送到哪个邮箱？

你：发到运维组邮箱

AI：明白了，我帮你创建：
- 时间：每天17:00
- 任务：分析日志
- 通知：发送到运维组邮箱
确认创建吗？

你：确认

AI：（创建成功）任务已创建！
```

### AI会主动问的问题

**不理解时：**
- "你是想查看数据还是导出报告？"
- "你说的'那个'是指哪个配置？"

**需要补充信息：**
- "请告诉我邮箱地址"
- "定时任务的执行时间是？"

**执行前确认：**
- "我理解你想XXX，对吗？"
- "确认要执行这个操作吗？"

## 🔧 浏览器要求

### 支持的浏览器

✅ **Chrome 25+** - 推荐
✅ **Edge 79+** - 推荐
✅ **Safari 14.1+** - 支持
⚠️ **Firefox** - 部分支持
❌ **IE** - 不支持

### 权限设置

首次使用需要授权：
1. 浏览器会提示"允许使用麦克风"
2. 点击"允许"
3. 如果误点了"拒绝"，需要手动开启：
   - Chrome: 地址栏左侧锁图标 → 网站设置 → 麦克风 → 允许
   - Edge: 同Chrome
   - Safari: 设置 → 网站 → 麦克风 → 允许

## ⚠️ 注意事项

### 隐私安全

- ✅ 语音识别在浏览器本地完成
- ✅ 只发送识别后的文字到服务器
- ✅ 不录制和存储语音文件
- ✅ 对话历史仅本地保存

### 识别准确度

**影响因素：**
- 环境噪音
- 说话清晰度
- 口音方言
- 麦克风质量

**建议：**
- 安静环境使用
- 使用外置麦克风
- 说话清晰标准
- 重要操作查看文字确认

### 网络要求

- 需要网络连接
- 建议使用稳定的WiFi
- 移动网络可能有延迟

## 🐛 故障排查

### 问题1：点击麦克风没反应

**原因：**
- 浏览器不支持
- 麦克风权限未授予
- 麦克风被占用

**解决：**
```bash
1. 使用Chrome/Edge浏览器
2. 检查麦克风权限
3. 关闭其他使用麦克风的程序
4. 刷新页面重试
```

### 问题2：识别不准确

**解决：**
- 说话清晰一些
- 靠近麦克风
- 减少环境噪音
- 使用标准普通话

### 问题3：AI没有回复

**原因：**
- 网络问题
- AI服务异常
- 消息太模糊

**解决：**
- 检查网络连接
- 查看浏览器控制台
- 换个说法重试

### 问题4：语音播报不工作

**原因：**
- 浏览器限制
- 声音被静音
- 系统音量关闭

**解决：**
- 检查系统音量
- 检查浏览器音量
- 打开"AI语音播报"开关
