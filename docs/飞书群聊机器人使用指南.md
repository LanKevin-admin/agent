# 飞书群聊机器人使用指南

## 🎯 快速开始

### 基本用法

```
@机器人 你的问题或指令
```

**示例：**
```
@机器人 帮我分析今天的日志
@机器人 创建定时任务：每天17点发送报告
@机器人 把邮箱改成 new@example.com
```

---

## ✨ 新特性：智能会话

### 特性1：连续对话，无需重复@

```
你：@机器人 帮我分析今天的日志    ← 首次必须@
机器人：（分析结果...）

你：再看看昨天的                  ← 不需要@
机器人：（昨天的分析...）

你：对比一下                      ← 不需要@
机器人：（对比结果...）
```

**规则：**
- ✅ 首次必须@机器人
- ✅ 后续5分钟内不需要@
- ⏰ 超过5分钟需要重新@

---

### 特性2：多个问题，快速提问

```
你：@机器人 问题1
你：@机器人 问题2
你：@机器人 问题3
```

**行为：**
- ✅ 每条都会收到回复
- ✅ 按时间顺序处理
- ✅ 10秒内开始响应

```

```


## ⚠️ 注意事项

### 1. 响应时间

- 机器人每**10秒**检查一次消息
- 发送后最多等待10秒
- 看到回复说明机器人正常工作

### 2. 会话过期

- 5分钟无交互会话自动过期
- 过期后需要重新@机器人
- 会话状态不影响功能使用

### 3. 消息格式

- 必须在群里@机器人
- 私聊暂不支持
- 文本消息为主

## 🔧 故障排查

### 问题1：@了但没有回复

**原因：**
- 机器人服务未启动
- 飞书配置错误
- 网络问题

**解决：**
1. 检查服务是否运行
2. 查看配置文件 `.env`
3. 运行测试：`python test_message_monitor.py`

### 问题2：后续消息没有回复

**原因：**
- 超过5分钟会话过期
- 消息超过1分钟前发送

**解决：**
- 重新@机器人
- 缩短消息间隔

### 问题3：回复内容被截断

**原因：**
- 飞书消息长度限制（4000字符）

**解决：**
- 分多次提问
- 使用更具体的问题
