# 飞书RPA监控系统 - 智能功能使用指南

## 🎯 新增功能概览

### 1. AI配置助手
不需要手动编辑配置文件，直接在智能对话中告诉AI：

**示例对话：**
```
你："帮我配置飞书，App ID是cli_xxx，App Secret是xxx，群组ID是oc_xxx"
AI："已成功更新飞书配置：FEISHU_APP_ID、FEISHU_APP_SECRET、FEISHU_CHAT_ID"

你："更新AI的API Key为sk-xxxxx"
AI："已更新AI配置"
```

AI会自动调用`update_config`工具写入`.env`文件。

---

### 2. 数据库支持
除了从飞书群读取日志，还能从数据库查询RPA记录：

**示例对话：**
```
你："从数据库查询今天的RPA日志，数据库配置是：
{
  "type": "mysql",
  "host": "localhost",
  "port": 3306,
  "database": "rpa_db",
  "user": "root",
  "password": "xxx",
  "table": "rpa_logs"
}"

AI会自动：
1. 连接数据库
2. 执行查询（自动生成SQL或使用你提供的SQL）
3. 提取字段（账号、状态、文件路径等）
4. 校验文件
5. 生成汇总报告
```

**字段映射示例：**
如果数据库字段名和标准字段不同，告诉AI：
```
你："字段映射是：账号名→account_name，运行状态→exec_status，文件路径→file_location"
```

---

### 3. 分析模板固化
当你对某次分析满意时，保存为模板，下次直接复用：

**第一次分析（成功）：**
```
你："从数据库查询今天的RPA记录，配置是{...}，字段映射是{...}"
AI："分析完成，成功率95%，生成报告xxx.txt"

你："很好！把这个分析流程保存为模板，名称是'生产环境日报'"
AI："已保存模板'生产环境日报'，包含数据库配置、SQL查询、字段映射"
```

**下次使用：**
```
你："使用生产环境日报模板分析今天的数据"
AI会自动：
1. 加载模板配置
2. 执行相同的查询
3. 使用相同的字段映射
4. 生成汇总报告
```

## 🛠️ 支持的数据库类型

- **MySQL** / MariaDB
- **PostgreSQL**
- **SQLite**

需要安装对应的Python驱动：
```bash
pip install pymysql        # MySQL
pip install psycopg2       # PostgreSQL
# SQLite 内置支持
```

## 🎨 前端页面导航

1. **仪表盘** (`/`) - 系统状态概览
2. **智能对话** (`/chat`) - 主要功能，自然语言交互
3. **分析模板** (`/templates`) - 查看和管理已保存的模板
4. **日志分析** (`/analysis`) - 表单式精确分析
5. **历史报告** (`/reports`) - 查看和下载历史报告
6. **配置管理** (`/config`) - 手动编辑配置（可选）

## 🚀 快速开始

1. 启动后端：`python web_server.py`
2. 启动前端：`cd web && npm run dev`
3. 打开浏览器：http://localhost:5175
4. 点击"智能对话"，开始配置！
