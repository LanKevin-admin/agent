## 🔍 测试失败排查

### 如果配置保存失败

**现象：** 提示"[Errno 2] No such file or directory"

**原因：** `.env`文件不存在且没有模板

**解决：**
1. 手动在EXE同目录创建`.env`文件
2. 或者确保`.env.example`被打包进EXE

**验证spec配置：**
```python
datas=[
    ...
    ('.env.example', '.'),  # ✅ 确保这行存在
]
```

### 如果定时任务不保存

**现象：** 创建任务后，任务页面看不到

**原因：** 数据库路径不对

**排查：**
1. 检查`data/rpa_monitor.db`是否在EXE同目录
2. 查看控制台是否有数据库错误
3. 使用SQLite查看器打开数据库，检查`scheduled_tasks`表

**手动测试：**
```bash
# 在EXE同目录打开PowerShell
python -c "from database.models import DB_PATH; print(DB_PATH)"
```

### 如果AI调用工具失败

**现象：** 对话报错"name 'false' is not defined"

**原因：** JSON解析问题未修复

**验证：**
```bash
# 检查修复是否生效
grep "json.loads" agent/skill_based_agent.py
```

## 🎯 完整打包命令

```bash
# 1. 构建前端
cd web
npm run build
cd ..

# 2. 清理旧文件
if exist build rmdir /s /q build
if exist dist rmdir /s /q dist

# 3. 打包
pyinstaller --clean feishu_rpa_monitor.spec

# 4. 验证
dir dist
echo 打包完成！
echo 输出文件：dist\飞书RPA监控系统.exe
```
